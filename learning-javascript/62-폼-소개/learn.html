데이터의 관문: 폼(Form)

사용자로부터 정보를 입력받고 전송하는 폼(Form)의 구조와 자바스크립트를 통한 제어 방법을 배웁니다.

폼 요소 및 필드 속성 요약

폼 구성 핵심 속성
속성/요소 역할 비고
method 정보 전송 방식 결정 GET(기본값), POST 등
action 데이터를 처리할 서버 URL 클라이언트 처리 시 생략 가능
label & id 필드 설명 및 상호작용 연결 웹 접근성 향상을 위해 필수
name 데이터 추출을 위한 키(Key) 서버나 JS에서 값 식별 시 사용

💡 학습 포인트

  • e.preventDefault(): 페이지 새로고침을 막고 JS로직을 먼저 실행합니다.
  • 폼 제출 방식: 버튼 클릭뿐만 아니라 input에서 Enter를 눌러도 이벤트가 발생합니다.
  • 데이터 처리: 처리 후 form.submit()을 호출하거나 비동기 통신을 수행합니다.
